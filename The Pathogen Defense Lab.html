Analysis Results:

Active Immunity

Your body generates its own antibodies after exposure to a pathogen or vaccine.

  • Long-lasting protection (memory cells)
  • Takes time to develop (days to weeks)
  • Examples: Vaccines, natural infection recovery

Passive Immunity

Receiving pre-made antibodies from an external source.

  • Immediate protection
  • Short-term effect (no memory cells)
  • Examples: Maternal antibodies, antivenom, antibody therapy

Technical Documentation: Perceptron Algorithm

Understanding how artificial neural networks mimic biological neural processing through a basic Perceptron implementation in C.

// Basic Perceptron Algorithm in C // Demonstrates how 'Weights' in AI relate to 'Synaptic Strength' in the brain #include <stdio.h> #include <stdlib.h> // Weights represent synaptic strength - how strongly one neuron influences another // In biology: Strong synapses = efficient signal transmission // In AI: High weights = stronger influence on output typedef struct { double *weights; // Analogous to synaptic strengths in biological neurons double bias; // Threshold for neuron activation (like action potential threshold) double learning_rate; } Perceptron; // Activation function - mimics the "all-or-nothing" firing of biological neurons // In biology: Neurons fire when stimulus exceeds threshold // In AI: Output is determined by weighted sum vs threshold int activate(double sum) { return (sum >= 0) ? 1 : 0; // Binary activation (like neuron firing or not firing) } // Initialize perceptron with random weights (like initial synaptic connections) void init_perceptron(Perceptron *p, int input_size) { p->weights = (double*)malloc(input_size * sizeof(double)); // Random initialization of weights (synaptic strengths) for (int i = 0; i < input_size; i++) { p->weights[i] = ((double)rand() / RAND_MAX) * 2 - 1; // Range: -1 to 1 } p->bias = ((double)rand() / RAND_MAX) * 2 - 1; p->learning_rate = 0.1; // How fast the network learns (plasticity rate) } // Predict function - simulates signal propagation through neuron // Biology: Dendrites receive inputs → Cell body integrates → Axon transmits output // AI: Inputs × Weights → Sum + Bias → Activation Function → Output int predict(Perceptron *p, double *inputs, int input_size) { double sum = p->bias; // Weighted sum - each input multiplied by its synaptic weight for (int i = 0; i < input_size; i++) { sum += inputs[i] * p->weights[i]; // Input signal × Synaptic strength } return activate(sum); // Neuron fires or doesn't fire } // Training function - implements Hebbian learning principle // "Neurons that fire together, wire together" - Donald Hebb // Strengthens connections (weights) that lead to correct outputs void train(Perceptron *p, double *inputs, int input_size, int target) { int prediction = predict(p, inputs, input_size); int error = target - prediction; // Calculate prediction error // Update weights based on error - synaptic plasticity in action! // If error > 0: strengthen connections (long-term potentiation) // If error < 0: weaken connections (long-term depression) for (int i = 0; i < input_size; i++) { p->weights[i] += p->learning_rate * error * inputs[i]; } p->bias += p->learning_rate * error; } // Main function - demonstrates the learning process int main() { Perceptron p; init_perceptron(&p, 2); // Training data for AND logic gate (example pattern recognition) double inputs[4][2] = {{0,0}, {0,1}, {1,0}, {1,1}}; int targets[4] = {0, 0, 0, 1}; // AND gate outputs // Training loop - repeated exposure strengthens correct pathways for (int epoch = 0; epoch < 1000; epoch++) { for (int i = 0; i < 4; i++) { train(&p, inputs[i], 2, targets[i]); } } printf("Training complete! Neural pathway established.\n"); free(p.weights); return 0; }

Key Biological-Computational Parallels:

In Biological Neurons:

  • • Dendrites receive input signals
  • • Synaptic strength determines signal impact
  • • Cell body integrates all signals
  • • Action potential threshold determines firing
  • • Synaptic plasticity enables learning

In Artificial Perceptron:

  • • Input variables receive data
  • • Weights determine input importance
  • • Weighted sum aggregates signals
  • • Bias + activation determines output
  • • Weight updates enable learning
